# iCloud Sorter

A Windows Desktop app for sorting photos from iCloud into folders.

## Installation
```bash
python -m venv .venv # or python3 if needed
source .venv/bin/activate  # Windows: .venv\Scripts\activate

pip install -e .
```

## Run locally
```bash
python -m app.main # or python3 if needed
```
